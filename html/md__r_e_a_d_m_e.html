v2.0
Galima atlikti unit testus.
instrukcija:
aplanke 'testai' galima sukurti testą pagal pavizdį.
terminale irasyti:

git clone https://github.com/google/googletest.git
cd googletest
mkdir build

tada atverti failą googletest/googletest/CMakeLists.txt
ten irasyti:
```
if(build-testus)
cxx_executable(mediana_unittest ../../testai gtest_main)
endif()
```
ir pridėti naują eilutę if'e:

cxx_executable('failo pavadinimas' ../../testai gtest_main)

Tada su terminalu nueit i '/googletest/build'
ir parašyti komandą: "cmake .. -DBUILD_GMOCK=OFF -Dgtest_build_samples=OFF -Dbuild-testus=ON"
Tada per Visual Studio atidaryti: "/googletest/build/googletest/gtest.sln"
ir ten paleisti testą(us).
